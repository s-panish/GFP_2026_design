## A. Logic tree (decision and reasoning path)

```
GOAL: six GFP-family sequences maximizing post-heat brightness  score = F_final / F_initial_WT
│
├─ [frame]            Score algebra: (F_init/F_init_WT)·(F_final/F_init) = F_final/F_init_WT.
│                     GFP brightness is stability-gated (Sarkisyan 2016 threshold), so BOTH
│                     factors are stability read-outs at two temperatures (initial, 72 °C).
│
├─ [decision · human] Anchor on sfGFP: robust fold ⇒ high F_initial + a large stability
│                     margin to spend on heat retention. (Designs may use any GFP parent;
│                     sfGFP chosen on these grounds.)
│
├─ AXIS 1 — initial brightness
│   ├─ [agent]            The only large GFP brightness DMS is avGFP (Sarkisyan, 51,715
│   │                     variants) → additive Ridge + isotonic. Held-out R² = 0.933.
│   ├─ [agent · footgun]  Numbering: the avGFP DMS drops the initial Met, so
│   │                     dataset_pos = protein_pos − 1. Verified (1810/1810 measured
│   │                     tokens match avGFP at offset −1); the pipeline ASSERTS this and
│   │                     refuses to run otherwise. A wrong offset silently flips signs.
│   └─ [agent + human]    Brightness drivers must be dual-validated (positive on BOTH the
│                         Ridge coefficient AND the single-mutant DMS brightness) →
│                         rejected the highest-coefficient mutations (e.g. D180E, K162T) as
│                         co-occurrence-inflated.
│
├─ AXIS 2 — heat retention
│   └─ [decision]         Negative surface supercharging (Thermal Green Protein recipe,
│                         Close 2015): surface K/N → E/D, brightness-neutral in the DMS,
│                         far from the chromophore. Exclude the epistatic pair
│                         N198D + K214E (measured −0.18 in DMS doubles).
│
├─ RELIABILITY SCREEN
│   └─ [agent]            Four independent signals per mutation: DMS support, additivity
│                         residual (non-additivity), burial (rSASA), chromophore distance.
│                         → flagged L44M (buried, ~7.4 Å) and replaced it with the clean
│                         surface driver Y237N.  (concrete design change)
│
├─ SELF-FALSIFICATION  (human-driven — the key rigor branch)
│   ├─ [human challenge]  "How come every other team is wrong but us?" → run the 20 past
│   │                     winners through our model as a falsification test.
│   ├─ [agent ERROR]      First diffed the winners vs sfGFP → inflated the UNSEEN count →
│   │                     wrongly concluded "the model is blind."
│   ├─ [human correction] "Are the past winners on avGFP?" → yes, they are avGFP-derived.
│   ├─ [agent corrected]  Re-scored vs the avGFP parent → the model SEES their mutations
│   │                     and still reads them as not bright (0/20 reach the sfGFP level)
│   │                     → they won on RETENTION, not initial brightness.
│   ├─ [agent discovery]  avGFP→sfGFP transfer is epistatic: S65T coef = −0.349 although
│   │                     sfGFP is brighter; the net background puts sfGFP AT the
│   │                     calibration ceiling → the model cannot predict "brighter than
│   │                     sfGFP," only "not dimmer." ⇒ score by DELTA relative to sfGFP.
│   └─ [human correction] An earlier "chromophore wall" claim was overstated → corrected
│                         to "high-variance region": deleterious mutations are enriched at
│                         chromophore-internal residues (Sarkisyan), yet the chromophore is
│                         also the engineer's best lever (S65T, T203Y, …). Tools blind to
│                         the chromophore get burned there; surface engineering is safe.
│
├─ CORRECTED, sfGFP-AWARE MODEL  [agent build]
│   └─ Verified+asserted offset · delta-not-absolute scoring · reference-residue check
│      (a mutation is scored directly only if its "from" residue matches both sfGFP and
│      avGFP) · coverage-aware fallback (only ~40% of single substitutions are measured;
│      unseen → position-level estimate + flag, never a silent zero) · honest confidence
│      flags (measured / estimated / ref-mismatch / low-support / buried+near-chromo) ·
│      brightness-only scope (does NOT predict 72 °C retention).
│
├─ COMPETITOR / TEAMMATE AUDITS  [agent analysis]
│   └─ Five directions scored on the corrected model → all predicted ≤ WT or dark, with
│      three recurring causes: numbering/reference errors, chromophore-region gambles, and
│      stability tools (FoldX / ThermoMPNN) that are blind to the chromophore and so are
│      pulled toward fluorescence-killing mutations.
│
├─ MERGE  (team decision across the two directions)
│   ├─ [decision]         One full workflow + the corrected brightness gate + the clean
│   │                     surface-validated sequences + GeneMeow's ColabFold figures and
│   │                     memory-safe packaging.
│   └─ [agent verify]     Applied the other direction's OWN gate criterion to GeneMeow's
│                         final 6 → all 6 RE-PICK (anchors L220V −0.166 buried/near-chromo;
│                         H148S unseen/near-chromo; Q183R −0.924). Our six pass cleanly →
│                         they are the merged submission.
│
└─ CONSOLIDATION  [agent]
    └─ Collapsed the modular code into one self-contained file `gfp_pipeline.py`;
       verified it produces byte-identical submission.csv and design_report.csv.
```

*Interpretation.* C2 shows the guards firing as intended: a chromophore proton-wire
mutation (S205Y) is read as strongly dimming; an unseen mutation falls back instead of
scoring zero; an invalid "from" residue is rejected and a position where sfGFP≠avGFP is
flagged. The same gate rejects the other direction's six candidates (they rest on the
numbering-bug/chromophore-region anchors L220V / H148S / Q183R), while our six pass. The
past-winners self-test confirms the model is not flattering us: it does not rate the
known winners as bright either — consistent with the honest conclusion that initial
brightness is a flat axis and the contest is decided on heat retention.

## I. Consolidated interpretation

The two agent branches converged on the same final principle: do not claim experimental improvement before the official assay. Instead, submit a rationally prioritized six-sequence portfolio with:

1. positive sfGFP-aware brightness delta evidence;
2. negative surface-charge / supercharging stability proxy;
3. preserved chromophore-forming motif;
4. high-confidence ColabFold structural validation;
5. clear human-in-the-loop correction history.

The final submission is therefore not presented as experimentally validated, but as a reproducible computational prioritization workflow optimized for the official dual-objective score.
